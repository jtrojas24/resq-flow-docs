# resq-flow-docs

Documentation site for `resq-flow`, built with React Router and Fumadocs.

## Development

```bash
bun run dev
```

## Checks

```bash
bun run types:check
bun run lint
bun run build
```

## Content

- `content/docs/` contains the docs content and navigation structure.
- `app/` contains the React Router app shell and Fumadocs integration.
