`resq-flow` is a local, flow-oriented observability app for understanding live execution and history across ResQ flows.

This site is the starting point for:

- local setup and day-to-day usage
- core product and flow concepts
- Graph V2 principles and layout direction
- integration guidance for `resq-agent` and other consumers
- troubleshooting relay, ingest, and docs-site issues
